wrapAcknowledgement(ack) → {function}

By default, Errors get serialized to empty objects when run through JSON.stringify. This function wraps an "acknowledgement" callback and checks if the first argument is an Error. If it is, that Error is serialized before being sent off to Socket.IO for serialization to be sent across the wire.
